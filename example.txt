[NAME]
Name of the item. This should be a single line describing the item.

[ID]
Generally generated by the 'new' command. Usually a uuid string but
any string without whitespaces is recognized.

[STATUS]

[ASSIGNED]

[DESCRIPTION]

[PRIORITY]
Priority of the item. Higher priority means higher priority.

[PARENT]
Each line indicates a parent item for this item. This is used to
organize the items when displayed as a tree.

Each lines should be an id. Additional text is allowed after the id
and is treated as a comment.

[LINK]
Same as PARENT but instead indicates another item of interest.

[PREREQ]

[ESTIMATE]

[TAG]

[TIME-REPORT]
